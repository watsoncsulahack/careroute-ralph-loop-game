# ChatDev-style Multi-Agent Plan (Applied)

Roles:
- ProductAgent: game objective, scope, win conditions
- NarrativeAgent: scenarios/events/dialog
- GameplayAgent: mechanics, balancing, scoring
- AndroidAgent: Kotlin screens/input/perf
- ArtPipelineAgent: keyframes/sprite sheets
- QAAgent: deterministic test scenarios

Protocol:
1. Spec handshake (confirm constraints)
2. Design debate (3 options)
3. Implementation assignment
4. Integration review
5. Test+polish pass
